# CineVerse AI — Entertainment DNA Engine

Every user receives a dynamic profile called **Entertainment DNA**.

This profile represents the user's entertainment preferences as weighted scores across genres and content types, and continuously evolves based on user interactions.

---

## Example Profile

| Category | Match Score |
|----------|-------------|
| Action | 95% |
| Sci-Fi | 91% |
| Anime | 88% |
| Adventure | 76% |
| Dark Themes | 82% |
| Comedy | 40% |
| Romance | 15% |

---

## How It Evolves

The Entertainment DNA updates automatically based on:

- Content rated by the user
- Items added to watchlists
- Search activity
- Click behavior
- Session and viewing patterns
- Mood selections

The more the user interacts with the platform, the more accurately the DNA reflects their true preferences.
