# CineVerse AI — Watchlists

Users can save, organize, and track content through a flexible watchlist system.

---

## Features

- Save content for later
- Organize content into collections
- Create custom named lists
- Track completed content

---

## Example Watchlists

| List Name | Purpose |
|-----------|---------|
| Weekend Watchlist | Content saved for the weekend |
| Best Anime Ever | Personal all-time favorites |
| Gym Music Collection | High-energy playlists for workouts |

---

## Premium Collections

Premium users can create unlimited collections and access enhanced organization features for managing large content libraries.
